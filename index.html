Crystal Collector

  1. A random target number will be generated at the start of the game.

  2. There are four crystals below. Clicking one will add a specific point value to your total score.

  3. You win the game if your total score matches the random target number. You lose the game if your total score goes above the random target number.

  4. The value of each crystal is hidden until it is clicked.

  5. Each time the game starts, the values of each crystal will change.
